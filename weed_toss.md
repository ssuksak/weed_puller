## 📌 게임 개요

**컨셉**: 도시에서 시골로 상경한 초보 농부가 밭의 잡초를 뽑는 매치 퍼즐 게임

**장르**: 캐주얼 매치 퍼즐 (애니팡/포코팡 스타일)

**플랫폼**: 앱인토스 (토스 인앱 미니게임) — HTML5 웹 기반

**타겟**: 토스 사용자, 캐주얼 게이머, 짧은 시간 즐기는 스낵 게임

---

## 🎮 핵심 메카닉

### 기본 조작
- **스와이프**: 같은 종류 잡초 3개 이상을 손가락으로 쓱~ 그으면 한꺼번에 뽑힘
- **폭탄 연타**: 폭탄잡초 등장 시 3번 빠르게 탭하여 해체 (실패 시 게임 오버)

### 게임 규칙
- 6x8 격자에 4종류 잡초가 가득 채워짐
- 뽑으면 위에서 새 잡초가 떨어져 채워짐 (낙하 시스템)
- 매치 없으면 자동 셔플
- 제한시간 30초 시작

### 시간 보상 (콤보)
| 수확 개수 | 시간 보너스 |
|-----------|-------------|
| 3개 | +1초 |
| 5개 | +2초 |
| 6개 | +3초 |
| 8개+ | +4초 |
- 최대 60초까지 누적 가능
- 잘하면 무한 플레이 가능!

### 폭탄 시스템
- 3번째 수확부터 새로 채워지는 잡초에 폭탄이 섞여 내려옴
- 시간이 갈수록 폭탄 수 증가 (최대 4개 동시)
- 타이머: 5초 → 최소 3초 (점점 짧아짐)
- 3탭으로 해체 / 실패 시 게임 오버

### FEVER 모드
- 20 콤보 달성 시 자동 발동 (8초간)
- 하늘이 주황색으로 전환
- BGM 빠른 버전
- 점수 배수 증가

- 모든 표정에 볼터치

## 🎵 사운드

### BGM (Web Audio API 생성)
- **시작 (30~16초)**: 동화풍 sine 멜로디 + 약한 드럼
- **중반 (15~6초)**: triangle 멜로디 + 베이스 깔림
- **막판 (5초↓)**: square 긴장 멜로디 + 강한 드럼
- **FEVER**: 빠른 템포 + square wave
- **폭탄 등장 시**: sawtooth 불안 멜로디 + 무거운 킥

### 효과음
- 수확: 뽁! (피치 상승)
- 큰 수확: 빅팝! (노이즈 버스트)
- 콤보: 콤보 수에 따라 피치 올라감
- 폭탄 탭: 피치 올라가는 팝
- 폭탄 해체: 빅팝 + 파티클
- 폭탄 폭발: 불협화음
- FEVER: 4음 팡파레
- 카운트다운: 비프
- 게임 오버: 하행 멜로디

## 🎨 비주얼

### 디자인 시스템
- **참고**: 토스 디자인 시스템 (TDS)
- **색상**: 토스 블루 #3182F6, 뮤트 파스텔 톤
- **폰트**: Jua (귀여운 한글 웹폰트)
- **border-radius**: 16~28px (토스 스타일 둥근 모서리)
- **그림자**: 부드럽고 다층 레이어

### 배경
- 하늘 + 구름 + 언덕 (원경)
- 밝은 잔디밭
- 밝은 베이지 밭 (흙)
- 밭 이랑 디테일
- 배경 반짝이 (별 떠다님)
- 이모지 장식 (🐛🐞🦋🐝🌻🌾🐔🐣)

### 이펙트
- 수확 파티클 (원형 + 별 모양, 골드/핑크/하늘색)
- 스크린 쉐이크
- 스와이프 3중 글로우 라인 (노란빛)
- FEVER 시 하늘 주황 전환
- 폭탄 빨간 펄스 글로우

## 🏆 랭킹 시스템

### 구조
- **DB**: Supabase (PostgreSQL)
- **인증**: 불필요 (anon key로 직접 REST API 호출)
- **테이블**: `weed_puller_rankings`

### 필드
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | BIGSERIAL | PK |
| nickname | TEXT(1~20) | 닉네임 (한글/영문/숫자만) |
| score | INTEGER(0~99999) | 점수 |
| max_combo | INTEGER | 최대 콤보 |
| survive_time | INTEGER | 생존 시간(초) |
| created_at | TIMESTAMPTZ | 등록일 |

### 보안
- RLS 활성화
- SELECT: 누구나 가능
- INSERT: 누구나 가능 (닉네임/점수 제약)
- UPDATE/DELETE: 완전 차단
- 점수 상한 99999
- 닉네임 XSS 방지 (정규식 제약)

### UI
- **시작화면**: TOP 5 랭킹 표시 (🥇🥈🥉)
- **결과화면**: 닉네임 입력 → 등록 → TOP 10 표시
- **공유**: Web Share API / 클립보드 복사

## 🛠 기술 스택

| 항목 | 기술 |
|------|------|
| 렌더링 | Canvas 2D |
| 오디오 | Web Audio API (실시간 생성) |
| 폰트 | Google Fonts (Jua) |
| DB | Supabase (PostgreSQL) |
| 호스팅 | GitHub Pages |
| 빌드 | 없음 (vanilla HTML/JS) |

### 파일 구조
```
weed-puller/
├── index.html     # UI + CSS
├── game2.js       # 게임 로직 + 렌더링 + 오디오
└── screenshots/   # 스크린샷
```

### 성능
- 총 파일 크기: ~40KB (gzip ~15KB)
- 외부 요청: Google Fonts 1개 + Supabase API
- 배경 캐시: 오프스크린 캔버스 (매 프레임 재렌더링 방지)
- 이모지 프리렌더링
- 60fps 안정 (모바일 포함)
